## Authentication

All API requests require authentication. See the [Authentication](/docs/reevit/openapi/authentication) section for details.

```bash
# API Key Authentication
curl -X GET "https://api.reevit.com/v1/payments" \
  -H "X-Reevit-Key: pfk_live_xxx" \
  -H "X-Org-Id: org_123"
```
